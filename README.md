# 💰 Nexus — Expense Intelligence

> A modern, full-featured expense tracking web application with real-time analytics, budget management, and financial insights.

🔗 Live Demo: https://sanduniapsara.github.io/expense-tracker/
🔗 **Source Code:** https://github.com/SanduniApsara/expense-tracker  

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 📊 Dashboard | KPI cards, spending flow charts, category breakdown |
| 💸 Expense Manager | Add, edit, delete expenses with categories, tags & payment method |
| 💰 Budget Tracker | Set limits per category with live progress & overspend alerts |
| 📈 Analytics | Bar, line, donut, heatmap & trend charts using Chart.js |
| 🔁 Recurring Expenses | Track subscriptions and repeated payments |
| 🔍 Smart Search | Live filtering by category, amount, and description |
| 😄 Mood Meter | Financial health indicator based on spending behavior |
| ✦ Insights Panel | Automatic analysis of spending patterns |
| 📱 Responsive UI | Mobile-friendly design with sidebar navigation |
| 💾 Data Storage | Persistent storage using localStorage API |

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

- Frontend: HTML5, CSS3 (Flexbox + Grid + Animations), Vanilla JavaScript (ES6+)
- Charts: Chart.js (Bar, Line, Donut, Heatmap, Trend)
- State Management: Custom localStorage-based store
- Architecture: Modular JavaScript (data / store / charts / ui / app)
- Optional Backend: Python Flask (for local development)
